# Changelog / 更新日志

All notable changes to this project are summarized here.

本文汇总项目的重要变化。

## Unreleased / 未发布

### Added / 新增

- Local-first C++20 / Qt6 / QML desktop workspace.
- SQLite content library, collection tasks, and run history.
- Jizhilia API catalog browsing and endpoint collection.
- Safe sample fallback when credentials are not configured.
- Markdown and XML export.
- CTK-style Provider, Exporter, and Analyzer registry.
- Linux, Windows, and Docker delivery scripts.
- QML control audit and DevPrompt alignment audit.
- Open-source documentation set: README, specification, architecture, developer guide, examples, contribution guide, security policy, and code of conduct.

- 本地优先 C++20 / Qt6 / QML 桌面工作台。
- SQLite 内容库、采集任务和运行历史。
- 极致了 API 目录浏览与 endpoint 采集。
- 未配置凭据时使用安全示例回退。
- Markdown 与 XML 导出。
- CTK 风格 Provider、Exporter、Analyzer 注册表。
- Linux、Windows、Docker 交付脚本。
- QML 控件审计和 DevPrompt 对齐审计。
- 开源文档集：README、规格、架构、开发指南、示例、贡献指南、安全策略和行为准则。
