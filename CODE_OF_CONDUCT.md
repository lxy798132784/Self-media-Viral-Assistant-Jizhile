# Code of Conduct / 行为准则

## Our standard / 我们的标准

We aim to keep this project practical, respectful, and focused on useful software.

我们希望这个项目保持务实、尊重和聚焦于有用的软件。

Examples of encouraged behavior:

鼓励的行为：

- Be clear and constructive.
- Explain trade-offs with evidence.
- Respect different skill levels and backgrounds.
- Keep discussions focused on the project.

- 清晰、建设性地表达。
- 用证据说明取舍。
- 尊重不同经验和背景。
- 讨论聚焦项目本身。

Examples of unacceptable behavior:

不接受的行为：

- Harassment, insults, or personal attacks.
- Publishing private information or secrets.
- Intentionally disruptive conduct.

- 骚扰、侮辱或人身攻击。
- 发布私人信息或密钥。
- 故意破坏协作。

## Enforcement / 执行

Maintainers may edit, close, or remove issues, pull requests, comments, and contributions that violate this code.

维护者可以编辑、关闭或移除违反本准则的 issue、PR、评论和贡献。
